Задание 1

Используя JavaScript Вам требуется создать простой таймер, который будет показывать обратный отсчет.
Для этого выведете в консоль строчку "Запускаем таймер" с помощью функции console.log. Затем добавьте 6 функций alert, каждая из которых будет выводить число, начиная с 5-ти и заканчивая нулем. После вывода нуля, напишите строчку кода, которая будет выводить в консоль "Обратный отсчет законен"



Задание 2
Тренируемся создавать переменные
Задача состоит в том, чтобы создать следующие переменные:
myName - присвоить переменной ваше имя
mySurname - присвоить переменной фамилию
myFavoriteDrink - указать любимый напиток
myFavoriteAnimal - любимое домашнее животное

При создании перемнных соблюдайте следующие условия:
1 - Используйте при объявлении перемнных только let и const. var мы использовать не будем так как он является устаревшим
2 - Выберите один из стилей напсиания кода: с точкой с запятой или без нее

После создания переменных их нужно вывести в консоль.
Для каждой перемнной должен быть свой console.log
